# Movie Review Sentiment Analysis App

This is a machine learning-based web application that predicts the sentiment of movie reviews as positive or negative. The model uses Natural Language Processing techniques, specifically TF-IDF vectorization, to convert text into numerical features and a trained classification algorithm to make predictions.

The application is built with Streamlit, providing a simple interface where users can input a review and receive real-time sentiment results. The trained model, vectorizer, and label encoder are saved and loaded to ensure fast and consistent predictions.
